== Installation ==

The general steps for setup are:

- Install Plugin
- Register for a free 7 day trial account on http://www.EZWebPlayer.com/
- Upload a video
- Enter your login information into the settings area for the plugin
- Post videos
- Pick a subscription plan and start uploading lots of videos!

There are 2 ways to post videos.  You can use the EZWebPlayer link in the bottom left of your blogs admin navigation. This will allow you to post many videos at once.

You can also use the Play Icon that is added to your toolbar when editing a post.  This allows you to insert videos directly into a post.
